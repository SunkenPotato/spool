# Spool
A dynamically typed, interpreted language written in Rust.

## Features
- [x] Bindings (a.k.a. Variables)
- [x] Mathematical Expressions
- [x] Code blocks
- [x] Binding references
- [x] CLI
- [ ] Order of Operations
- [ ] Booleans
- [ ] Functions
- [ ] Structs
  - [ ] Associated methods
- [ ] File interpreting
